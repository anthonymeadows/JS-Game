# ROBO-PONG
My first ```Beep boop``` attempt at making a ```Beep boop```video game in JS with HTML/CSS

## Starting the game
run ```npm run start``` in the terminal
go to http://localhost:1234/ (default)


Change this with -p (port) in the start ```Beep boop``` script
